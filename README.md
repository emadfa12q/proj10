# Work Time Tracker Android v8

نسخه شخصی اندروید برای ثبت فعالیت برنامه‌ها، گزارش‌گیری، اسکرین‌شات اختیاری با Accessibility، تایم‌لاین زنده و محاسبه بیکاری/توقف.

## تغییرات v8

- قابلیت **زمان بیکاری/توقف** دوباره اضافه شد.
- در صفحه خانه، کارت‌های **زمان فعال** و **بیکاری/توقف** نمایش داده می‌شوند.
- در گزارش‌ها، **جمع فعال** و **بیکاری بازه** جداگانه نمایش داده می‌شود.
- در خروجی PDF، زمان بیکاری هم کنار جمع فعال نوشته می‌شود.
- در تنظیمات، بخش **بیکاری و توقف زمان** اضافه شد.
- کاربر می‌تواند حد توقف را به دقیقه تنظیم کند؛ مقدار پیش‌فرض ۱۰ دقیقه است.
- مقدار `0` یعنی محاسبه/توقف خودکار بیکاری خاموش باشد.
- اگر Screenshot Accessibility فعال باشد، تعامل‌های کاربر برای تشخیص بیکاری دقیق‌تر استفاده می‌شود.
- اگر در مدت تعیین‌شده تعامل جدیدی ثبت نشود، فعالیت جاری بسته می‌شود و فاصله به‌عنوان بیکاری حساب می‌شود.

## سایر قابلیت‌های حفظ‌شده

- قفل عددی ورود با رمز پیش‌فرض `1183`
- تغییر رمز ورود در تنظیمات
- ثبت زنده فعالیت‌ها هر ۱ ثانیه
- نمایش فعالیت زنده با برچسب زنده
- اسکرین‌شات اختیاری با دکمه نمایش اسکرین‌شات، بدون پیش‌نمایش داخل کارت
- پاک‌سازی خودکار اسکرین‌شات‌ها بر اساس ۳ روز، ۱ هفته، ۱ ماه یا همیشه نگه دار
- انتخاب نام و محل ذخیره PDF با فایل‌پیکر اندروید
- تایم‌لاین با دکمه‌های + و − برای زوم
- UI ریسپانسیو با فاصله امن از Status Bar و Navigation Bar

## ساخت APK در GitHub Actions

فایل‌ها را مستقیم در ریشه repository قرار بده. این موارد باید مستقیم دیده شوند:

```text
app
.github
build.gradle
settings.gradle
```

بعد از commit، از تب Actions، workflow با نام **Build Android APK** را اجرا کن.
